# Solux Notes

App de notas ligera y privada de Solux OS (Python 3 + GTK 3).

- Notas en **Markdown plano** en `~/.local/share/solux-notes/`.
- Lista lateral con **búsqueda**, **autoguardado** y creación/borrado rápido.
- **Sin nube, sin cuentas, sin telemetría** — tus notas viven solo en tu disco.
- Tema oscuro Solux, editor monoespaciado.

## Ejecutar
```bash
python3 solux_notes.py
```

## Instalación en el sistema
Se copia a `/usr/share/solux/notes/` con lanzador `/usr/bin/solux-notes` y
`solux-notes.desktop` en `/usr/share/applications/`.
